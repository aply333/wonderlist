get /users

* gets all users

post /users
* inserts 1 user

get /users/:id
* gets 1 users info 

get /users/:id/todos
* gets 1 users todo items 

post /users/:id/todos
 * creates 1 todo item

 * gets all todos 

 get /todos/:id 
 * gets ID'd todo

put /todos/:id 
 * updates ID'd todo

 * removes ID'd todo
